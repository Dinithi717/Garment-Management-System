# Garment-Management-System
Garment Management System
Skills & Tools: Java (Spring Boot), MS SQL Server, Spring Data JPA, Sprint Planning, Inventory Management.

Objective: To digitize and streamline the end-to-end inventory and administrative workflows for a textile manufacturing environment.

Engineered a specialized Inventory Management Module to track raw materials and finished goods with real-time stock level updates.

Developed a secure backend using Spring Boot, implementing RESTful APIs for efficient data handling and system scalability.

Designed and managed a relational database schema in MS SQL Server to maintain data integrity across complex textile production records.

Collaborated through structured Sprint Planning, acting as the Inventory Manager persona to ensure the system met practical industry requirements.
